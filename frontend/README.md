# Brisk - Web UI

## Tech Stack

- Vite 7.3.0
- React 19.2.3
- React Router v7.11.0
- Tailwind CSS 4.1.18
- Zustand 5.0.9
- shadcn/ui + Radix UI
- TypeScript 5.9.3
- Biome 2.3.10

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

```bash
npm install
npm run dev
```

Dev server runs at http://localhost:3000
